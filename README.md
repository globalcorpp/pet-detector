# Pet-detector

In this simple project, I implemented basic concepts of AI.
I used [https://teachablemachine.withgoogle.com/](https://teachablemachine.withgoogle.com/) to train the model.
I also used a dataset from [https://www.kaggle.com/](https://www.kaggle.com/).

In this project, you can either upload an image or use a webcam to process it and get the final prediction as a percentage.

For simplicity, I used only two pet animals: cats and dogs.

I used AI assistance to improve the website layout and rearrange the elements

You can't run this app locally because of CORS. Instead, use the Live Server extension in VS Code

Feel free to improve it.
